# sealdice-ui

> 海豹核心的前端实现

## 说明

本仓库为海豹核心的前端实现，使用 Vue3 + ElementPlus + tailwindcss 开发。

后端仓库：[sealdice-core](https://github.com/sealdice/sealdice-core)

此仓库的 Issue 已关闭，请在后端仓库中提交 Issue。包括与前端相关的 Issue。

## 部署

```bash
npm install
```

### 开发环境

```bash
npm run dev
```

### 生产环境

```bash
npm run build
```
